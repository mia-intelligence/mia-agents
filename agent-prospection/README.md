\# Agent Prospection MIA



\## Objectif



Identifier automatiquement deux types de cibles sur les départements 

83, 13 et 84 (sud Luberon) :

\- Centres de formation pour proposer les formations MIA

\- Entrepreneurs et artisans comme prospects pour l'offre d'automatisation



\## Flux de fonctionnement



1\. Définition des critères de recherche (secteur, zone géographique)

2\. L'agent interroge plusieurs sources de données en parallèle :

&#x20;  - Apollo (base de données entreprises)

&#x20;  - Pappers (données légales entreprises françaises)

&#x20;  - data.gouv (données publiques officielles)

&#x20;  - Google (recherche complémentaire)

3\. Consolidation des résultats dans une liste de prospects qualifiés

4\. Exploitation de la liste pour prise de contact



\## Outils utilisés



\- Claude Code (Anthropic)

\- Apollo API

\- Pappers API

\- data.gouv API

\- Google



\## Périmètre géographique



\- Var (83)

\- Bouches-du-Rhône (13)

\- Sud Luberon — Vaucluse (84120)
