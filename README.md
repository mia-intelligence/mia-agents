Documentation des agents IA construits par MIA — Médiation Intelligence Appliquée



\# MIA — Agents IA construits par Laetitia Coloré



\## Présentation



MIA (Médiation Intelligence Appliquée) est une structure d'accompagnement 

et de formation à l'IA pour les TPE, artisans et commerçants.



Ce repository documente les agents IA conçus et déployés par Laetitia Coloré 

entre janvier et avril 2026, dans le cadre du développement de MIA.



\## Agents documentés



| Agent | Fonction | Statut |

|-------|----------|--------|

| \[Agent Communication](./agent-communication/) | Création automatisée de contenus blog, Facebook, Instagram | ✅ En production |

| \[Agent Prospection](./agent-prospection/) | Identification de prospects et centres de formation | 🧪 En test |

| \[Agent Veille](./agent-veille/) | Veille hebdomadaire automatisée sur l'IA | ✅ En production |
